# GalWriter AI v1.2.5

> このリリースでは、GalWriter AI のローカル制作体験を重点的に改善しました。プロジェクトをローカルに保存できるようになり、AI Provider 設定と API Key はプロジェクトエクスポートに含まれません。デスクトップアプリではより完全な動画書き出し経路を提供し、Web 版では安定した直接書き出し形式として WebM を維持しつつ、MP4 / MOV / MKV が必要なユーザーをデスクトップ版へ案内します。

**[中文版 →](https://github.com/Mingwen-Cui/GalWriter/blob/main/docs/release-notes/v1.2.5-zh.md) | [English →](https://github.com/Mingwen-Cui/GalWriter/blob/main/docs/release-notes/v1.2.5-en.md)**

## 概要

GalWriter AI v1.2.5 は、エディタを「一時的な Web ツール」から「継続的に制作できるローカルプロジェクトワークスペース」へ近づけるリリースです。

このバージョンでは、ローカルプロジェクト管理、より安全な AI Provider 設定、動画書き出し体験の 3 つを中心に改善しました。ユーザーはプロジェクト一覧から作成、インポート、オープン、名前変更、削除ができます。AI 設定はローカルに保存され、API Key がプロジェクトファイルに混入することを防ぎます。デスクトップ版は内蔵 FFmpeg によりより完全な動画書き出しに対応し、Web 版は安定した直接書き出し形式として WebM を継続します。

---

## ハイライト

- **ローカルプロジェクトライブラリ**：新規作成、インポート、最近使ったプロジェクトからの再開に対応。
- **より安全な API Key 保存**：AI Provider 設定はローカルに保存され、プロジェクト ZIP には含まれません。
- **プロジェクト単位の AI 会話**：AI 会話は現在のプロジェクト内にのみ保存されます。
- **統一された AI 設定**：Text、Image、Voice AI それぞれで複数の Provider 設定を保存できます。
- **動画書き出し案内の改善**：Web 版で MP4 / MOV / MKV を選ぶと、ブラウザ標準 alert ではなくアプリ内のデスクトップ版案内を表示します。
- **デスクトップ版 FFmpeg 書き出し経路**：App 端では MP4、MOV、MKV、WebM などの一般的な形式に対応します。
- **プロジェクトドキュメント改善**：中国語、英語、日本語 README をオープンソースプロジェクトのトップページとして整理しました。

## 追加

### ローカルプロジェクト管理

最近使ったプロジェクトの作成、インポート、オープン、削除、名前変更を行うプロジェクト選択ウィンドウを追加しました。

エディタに入った後、最近使ったプロジェクトから制作を再開できます。プロジェクトがない場合は、新規作成または既存プロジェクトのインポートが可能です。AI 会話、シナリオノード、プロジェクト設定は現在のプロジェクトに保存されます。

### AI Provider 設定センター

統一された AI Provider 設定管理を追加しました。

- **Text AI**：AI アシスタント、続き生成、推敲、分析など。
- **Image AI**：キャラクター、シーン、背景画像生成。
- **Voice AI**：ナレーション、吹き替え、テキスト読み上げ。

各種類で複数の設定を保存し、現在有効な設定を選択できます。設定名、Provider、Model、API URL、API Key はローカルに保存されます。

### 動画書き出し形式の方針

デスクトップアプリでは FFmpeg により一般的な動画形式を書き出せます。Web 版はブラウザ制限により、直接書き出しは WebM のみです。

Web ユーザーが MP4、MOV、MKV を選択した場合、GalWriter AI は統一されたアプリ内案内を表示し、デスクトップ版のリリースページへ誘導します。

## 変更

### プロジェクト保存と復元ロジック

通常保存したプロジェクトを再度開いたときにも「前回の進行状況を復元しますか」と聞かれる不自然な挙動を修正しました。

復元確認は、未保存の autosave 下書きが存在する場合のみ表示されます。正常に保存されたプロジェクトでは、再オープン時に復元確認は表示されません。

### プロジェクトタイトル編集

プロジェクト名はエディタ Header から編集できるようになりました。空のタイトルで保存する場合は、デフォルトのプロジェクト名を自動生成します。

### モーダルスタイルの統一

削除確認、AI 会話削除、Web 動画形式案内などは、可能な限りブラウザ標準 alert / confirm ではなくアプリ内 UI モーダルを使用します。

### 最近使ったプロジェクトのスクロール

最近使ったプロジェクトが多い場合、プロジェクトウィンドウ内部でスクロールし、モーダル全体が伸び続けないようにしました。

## 修正

- 保存成功後に待機中の autosave タイマーが古い下書きを書き戻す可能性を修正。
- Web 版でプロジェクトを閉じる、または再オープンした際に復元確認が誤って表示される問題を修正。
- プロジェクト選択ウィンドウが意図せず閉じる問題を修正。
- プロジェクト名が長い場合の Header レイアウト問題を修正。
- ユーザーが AI 設定を持っていない場合に、不要な空設定が表示される問題を修正。
- 一部の TypeScript 型推論問題を修正。
- Web 版で WebM 以外の形式を選んだ際にブラウザ標準 alert が表示される問題を修正。

## アップグレードメモ

### API Key はエクスポートされません

このバージョンから、AI 設定と API Key は現在の端末にのみ保存されます。エクスポートされたプロジェクトファイルには API Key は含まれません。

別の端末で使用する場合は、その端末で AI Provider 設定を再作成してください。

### Web 版の動画書き出し制限

Web 版で直接書き出せる形式は WebM のみです。MP4、MOV、MKV が必要な場合はデスクトップ版を使用してください。

### Windows Defender の警告

アプリはまだコード署名されていないため、Windows Defender が未知の発行元として警告する場合があります。実行前に、公式 GitHub Releases から取得したファイルであることを確認してください。

---

## 既知の制限

- このリリースは主に Windows ユーザー向けです。
- Web 版動画書き出しで保証される形式は WebM のみです。
- AI Provider の利用可否、クォータ、課金、データポリシーはユーザーが設定した各サービスに依存します。
- ローカル Stable Diffusion WebUI を使用する場合、ユーザー自身で WebUI を起動し API を有効にする必要があります。
