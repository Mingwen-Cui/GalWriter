# GalWriter AI v1.2.5

> 本次版本重点完善 GalWriter AI 的本地创作体验：项目可以保存在本机，AI 接口配置与 API Key 不会进入项目导出；桌面端提供更完整的视频导出路径，Web 端则保持 WebM 作为稳定导出格式，并引导需要 MP4 / MOV / MKV 的用户下载 App 端。

**[English →](https://github.com/Mingwen-Cui/GalWriter/blob/main/docs/release-notes/v1.2.5-en.md) | [日本語 →](https://github.com/Mingwen-Cui/GalWriter/blob/main/docs/release-notes/v1.2.5-ja.md)**

## 概览

GalWriter AI v1.2.5 将编辑器从“临时网页工具”进一步推进到“可以持续创作的本地项目工具”。

这一版围绕三个方向做了大量改进：本地项目管理、AI 接口配置安全、以及视频导出体验。用户可以在项目列表中创建、导入、打开、重命名和删除项目；AI 配置会保存在当前设备本地，避免 API Key 混入项目文件；桌面端新增 Full / Lite 两种安装包策略，Full 版内置精简 FFmpeg，Lite 版保持更小体积；Web 端继续使用更稳定的 WebM 导出。

---

## 亮点

- **本地项目库**：打开软件后可以创建新项目、导入项目或继续最近项目。
- **更安全的 API Key 存储**：AI 接口配置保存在本机，不会随项目 ZIP 导出。
- **项目内 AI 对话隔离**：一个项目中的 AI 对话只保存在该项目中。
- **统一 AI 接口设置**：支持文本、图片、语音 AI 多套配置，并可切换当前使用配置。
- **视频导出体验优化**：Web 端选择 MP4 / MOV / MKV 时会提示使用桌面版，不再弹浏览器原生 alert。
- **桌面端 Full / Lite 安装包**：Full 版内置精简 FFmpeg，推荐大多数用户；Lite 版不内置 FFmpeg，适合在意体积或已有 FFmpeg 的用户。
- **桌面端 FFmpeg 检测**：App 端会优先检测内置 `ffmpeg.exe`，再检测应用目录和系统 PATH，找不到时用应用内窗口提示。
- **开源项目门面优化**：中、英、日 README 已按开源项目首页结构重写，方便新用户理解项目定位。

## 新增

### 本地项目管理

新增项目选择窗口，用于创建、导入、打开、删除和重命名最近项目。

用户进入编辑器后，可以从最近项目继续创作；如果没有项目，也可以创建新项目或导入已有项目。项目中的 AI 对话、剧情节点和项目设置会跟随当前项目保存。

### AI 接口配置中心

新增统一的 AI 接口配置管理：

- **文本 AI**：用于 AI 助手、续写、润色、分析等。
- **图片 AI**：用于角色图、场景图、背景图生成。
- **语音 AI**：用于朗读、配音和文本转语音。

每种类型都可以保存多套配置，并选择当前启用的配置。配置名称、Provider、Model、API URL 和 API Key 都会保存到本机。

### 视频导出格式策略

桌面 App 端支持通过 FFmpeg 导出常见视频格式。Full 版安装包会内置 `ffmpeg.exe`，Lite 版不内置 FFmpeg；如果用户选择 Lite 版，MP4、MOV、MKV 等格式需要电脑上已有 FFmpeg。Web 端由于浏览器限制，只直接支持 WebM。

当 Web 用户选择 MP4、MOV 或 MKV 时，应用会显示统一风格的提示窗口，并引导下载桌面版。

### 桌面安装包拆分

新增两种桌面端发布资产：

- `GalWriter-Setup.exe`：Full 版，内置精简 FFmpeg，推荐下载。
- `GalWriter-Setup-Lite.exe`：Lite 版，不内置 FFmpeg，体积更小；MP4 / MOV / MKV 导出需要用户电脑已有 FFmpeg。
- `GalWriter-AI-v1.2.5-windows-x64.exe`：便携 Lite 版，不需要安装，双击即可运行。
- `GalWriter-AI-v1.2.5-windows-x64-portable-full.zip`：便携 Full 版，解压后双击运行，并保留同目录下的 `ffmpeg.exe`。

构建脚本只会在 Full 版中携带 `ffmpeg.exe`，不会打包 `ffprobe.exe`、文档、presets 等无关文件。

## 改进

### 项目保存与恢复逻辑

修复“正常保存后再次打开仍提示恢复上次进度”的反直觉体验。

现在只有在存在未保存 autosave 草稿时才会提示恢复；如果用户已经正常保存项目，再次打开不会重复询问。

### 项目标题编辑体验

项目标题改为在编辑器 Header 中编辑。空标题保存时会自动生成默认项目名，避免项目列表出现无意义空名称。

### 弹窗风格统一

删除确认、AI 对话删除、Web 视频格式提示等场景尽量使用应用内 UI 弹窗，不再依赖浏览器原生 alert / confirm。

### 最近项目滚动体验

当最近项目较多时，项目窗口支持内部滚动，不会继续把整个窗口拉长。

## 修复

- 修复项目保存后 autosave 定时器可能写回旧草稿的问题。
- 修复 Web 端关闭或重新打开项目时误触发恢复提示的问题。
- 修复项目选择窗口被自动关闭的问题。
- 修复项目名过长时 Header 布局异常的问题。
- 修复 AI 配置列表在用户没有配置时错误出现默认空配置的问题。
- 修复部分 TypeScript 类型推断问题。
- 修复 Web 视频导出选择非 WebM 时使用浏览器原生提示的问题。

## 升级说明

### API Key 不会进入项目导出

从这一版开始，AI 配置和 API Key 只保存在当前设备本地。导出的项目文件不会包含 API Key。

如果你换电脑使用，需要在新设备上重新创建 AI 接口配置。

### Web 端视频导出限制

Web 端只直接导出 WebM。需要 MP4、MOV 或 MKV 的用户，请下载桌面 App。

### 桌面 App Full / Lite 选择

如果希望“一键安装、点击即用”，请下载 `GalWriter-Setup.exe` Full 版。如果更在意安装包体积，并且电脑已经有 FFmpeg，可以下载 `GalWriter-Setup-Lite.exe`。

### Windows Defender 提醒

如果 Windows Defender 提示未知发布者或拦截下载，通常是因为应用还没有代码签名。请确认文件来自官方 GitHub Releases 后再运行。

## 已知限制

- 当前发布主要面向 Windows 用户。
- Web 端视频导出只承诺 WebM。
- Lite 版桌面 App 不内置 FFmpeg，非 WebM 视频格式依赖用户电脑上的 FFmpeg。
- AI Provider 由用户自行配置，模型权限、余额、请求限制和数据政策取决于对应服务商。
- 若使用本地 Stable Diffusion WebUI，需要用户自行启动 WebUI 并开启 API。

| 文件                                         | 说明                                                                 |
| -------------------------------------------- | -------------------------------------------------------------------- |
| `GalWriter-Setup.exe`                        | Full 版，内置精简 FFmpeg，推荐大多数用户下载                         |
| `GalWriter-Setup-Lite.exe`                   | Lite 版，不内置 FFmpeg，体积更小；非 WebM 视频导出需要系统 FFmpeg    |
